# 요청 요약

- README를 청중용 발표 자료처럼 간략하고 시각적으로 보이게 수정한다.
- 긴 설명형 문장을 줄이고, 함수 기준 전체 실행 흐름 섹션은 제거한다.

# 결정 사항

- README를 표, mermaid 다이어그램, 짧은 예시 중심으로 재구성한다.
- 함수 호출 상세 설명은 제거하고 시스템 구조와 핵심 구조체, 팀 포인트만 남긴다.
- 시연용 과장 문장이나 부연 설명은 넣지 않는다.

# 현재 브랜치 상태

- `simplify` 브랜치에서 작업했다.
- 작업 시작 시 추적되지 않은 `demo-data/` 디렉터리가 있었다.
