# 요청 요약

- README에서 구조체 관계도는 제거한다.
- 실행 흐름은 실제 코드 기준 `main(argc, argv)`와 `parse_arguments()` 흐름으로 바꾼다.
- 실행 흐름 설명은 긴 문단 대신 간단한 표와 시각 자료 중심으로 정리한다.
- 시연 예시는 이미지 파일로 교체한다.

# 결정 사항

- 실행 흐름은 현재 코드 기준인 `argc == 6` 파일 실행 모드로 표현한다.
- 시연 예시는 저장소 안의 SVG 이미지 자산으로 추가한다.

# 현재 브랜치 상태

- `simplify` 브랜치에서 작업했다.

# 완료한 작업

- `README.md`에서 구조체 관계도 다이어그램을 삭제했다.
- 실행 흐름을 `main(argc, argv) -> parse_arguments() -> run_program()` 기준 다이어그램으로 수정했다.
- 실행 흐름 섹션을 표와 단계별 mermaid 예시 이미지 중심으로 정리했다.
- `docs/images/demo-run.svg`를 추가하고 README 시연 예시에 연결했다.

# 리뷰 결과

- 실행 흐름이 실제 코드와 더 직접적으로 연결되도록 정리됐다.
- 시연 예시가 텍스트 블록보다 발표 자료에 가까운 이미지 형태로 바뀌었다.
